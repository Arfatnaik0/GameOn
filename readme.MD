## API Overview

### Games
- `GET /games/search`
- `GET /games/featured`
- `GET /games/popular`
- `GET /games/{id}`
- `GET /games/{id}/screenshots`

### Reviews
- `GET /reviews/game/{id}`
- `GET /reviews/`
- `POST /reviews`
- `PUT /reviews/{id}`
- `DELETE /reviews/{id}`

### Users
- `GET /users/me`
- `PUT /users/me`
- `DELETE /users/me`
- `GET /users/{id}`

### Lists
- `GET /lists/me`
- `POST /lists`
- `PUT /lists/{game_id}`
- `DELETE /lists/{game_id}`
